# Gaz Express Conversion OS

Status: Draft scaffold

Gaz Express is a placeholder brand name for an LPG supplier campaign targeting restaurants and hospitality businesses.

This folder contains the doctrine, context, memory and PRD scaffold for a prospect conversion system.

The system is not just a landing page. It is a sales conversion loop:

Representative visit -> WhatsApp introduction -> mobile landing page -> trust building -> WhatsApp conversation -> sales follow-up -> quote -> customer.

## Core Files

- `GAZ_EXPRESS_DOCTRINE.md` — strategic doctrine and non-negotiable rules.
- `CONTEXT.md` — current project context for agents.
- `MEMORY.md` — stable decisions and campaign memory.
- `PRD_INDEX.md` — product requirement document index.
- `prd/PRD_008_CONVERSATION_CAPTURE_ENGINE.md` — revised WhatsApp-first conversion PRD.

## Operating Principle

Do not position Gaz Express as only an LPG supplier.

Position the company as a kitchen continuity partner for restaurants and hospitality operators.

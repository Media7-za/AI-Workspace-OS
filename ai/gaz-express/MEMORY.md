# Gaz Express Memory

Status: Draft
Project: Gaz Express Conversion OS

This file records stable project decisions so future agents do not drift.

## Stable Decisions

### Decision 001 — Positioning

Gaz Express must be positioned as a kitchen continuity partner, not merely an LPG supplier.

### Decision 002 — Target Market

The campaign targets restaurants and hospitality businesses.

### Decision 003 — Sales Context

The representative has usually already made contact before the digital journey begins.

The campaign is responsible for converting warm/cold contact details into conversation and trust.

### Decision 004 — Landing Page Role

The landing page exists to build credibility and confidence.

It is not a generic brochure page.

### Decision 005 — WhatsApp-First Conversion

PRD 008 must loop back into WhatsApp.

The primary conversion action is to start a WhatsApp conversation, not submit a long form.

### Decision 006 — Follow-Up Engine

A follow-up engine is required because most prospects will not convert on day one.

Follow-up is part of the conversion system, not an optional sales task.

### Decision 007 — Hero Message

The strongest current campaign message is:

KEEP YOUR KITCHEN RUNNING.

### Decision 008 — Core Fear

The campaign must address the fear of running out of LPG during service.

### Decision 009 — Proof Before Price

The campaign should build trust before pushing pricing.

### Decision 010 — Asset Stack

The current asset stack is:

- WhatsApp graphic
- mobile-first landing page
- company profile PDF
- WhatsApp quote request
- follow-up sequence
